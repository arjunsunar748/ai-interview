# 🚀 Overview

> **AI-Powered Mock Interview & Career Preparation System** is a **fully offline**, production-ready interview preparation platform that simulates real interview experiences using **Local LLMs**, **Speech Recognition**, **Computer Vision**, and **AI-based Cheating Detection**.

No OpenAI API.

No Internet Required.

Everything runs **locally** using **Ollama + Whisper + FastAPI + React**.
